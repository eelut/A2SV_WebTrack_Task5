# Contact Form Project

A simple and responsive **Contact Form built with React and TypeScript** using `react-hook-form` for state management, validation, and submission handling.  

---

##  Features

- Form fields:
  - **Name** (required)
  - **Email** (required, validated for proper email format)
  - **Message** (required)
- **Form validation** with clear error messages
- **Form submission handling**:
  - `isSubmitting` state for disabling submit button while submitting
  - Automatic form reset on successful submission (`reset()`)
- Clean and responsive UI  
- Fully typed with TypeScript  

---

## Tech Stack

- **React** (Functional Components & Hooks)  
- **TypeScript**  
- **React Hook Form**  
- **CSS** for styling  
- **Vite** as the development server and build tool

## Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/eelut/A2SV_WebTrack_Task5.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```
